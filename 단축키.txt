참고함

Ctrl + B : 왼쪽 탐색기 창 끄기/켜기
Ctrl + '+' / Ctrl + '-' : 폰트 사이즈 조절
Ctrl + Pageup / Pagedown : 열어놓은 탭 왔다갔다 하기
Ctrl + Tab : 탭 이동


Ctrl + / : 주석달기 (드레그하면 여러줄도 가능)
Alt + Shift + a : 블록 주석 묶기
    - 주석으로 묶을 코드 부분을 드래그하고 단축키를 누르면 주석 처리 된다.
Ctrl + F : 일치하는 텍스트 찾기
Ctrl + H : 일치하는 텍스트 바꾸기
Alt + 위아래 : 해당 줄에 있는 코드 위 아래로 이동
Shift + Alt + (↑, ↓) : 한줄 복사(현재줄의 코드가 위 아래에 똑같이 복사됨)
Ctrl + L : 라인 전체 선택
Ctrl + x : 한줄 삭제
Ctrl + D : 같은 단어 하나씩 선택 (밑으로 하나씩 선택됨)
Ctrl + Shift + L : 같은단어 전체 선택
Alt + Click : 멀티 커서
    - 멀티커서 후 타자치면 해당 커서에 전부 타자됨
Ctrl + Alt + (↑, ↓) : 길다란 커서
    - 밑으로 내리면 일자로 멀티커서 되고 삭제, 변경이 동시에 됨
Alt + Shift + 마우스 드래그 : 자유 영역 지정
    - 사각형모양으로 블록지정되고 동시 삭제, 변경
Alt + Shift + I : 드래그된 모든 줄의 끝에 커서를 모두 생성


Shift + Alt + F / 전체 선책후 Ctrl + K + F : 자동 정렬
    - 단축키를 누르면 자동 들여쓰기 및 스타일 포맷을 한 번에 맞출 수 있습니다.
    - 매우 많이 사용되는 플러그인(plug-in)입니다.
